# Rock Paper Scissors

A classic Rock Paper Scissors game implementation built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## 📋 Project Description

This project implements a fully functional Rock Paper Scissors game where players can compete against the computer. The game features an interactive user interface with score tracking and game logic implementation.

## 🎮 Features

- Play Rock Paper Scissors against the computer
- Real-time score tracking
- User-friendly interface
- Responsive design
- Reset game functionality

## 🛠️ Technologies Used

- **JavaScript** (55.6%) - Game logic and interactivity
- **HTML** (19%) - Structure and markup
- **CSS** (25.4%) - Styling and layout

## 🚀 Getting Started

### Prerequisites
- A modern web browser

### Installation

1. Clone the repository:
   ```bash
git clone https://github.com/Garychamp/the-odin-project-project-rock-paper-scissors.git
cd the-odin-project-project-rock-paper-scissors
```

2. Open `index.html` in your web browser

## 📖 How to Play

1. Choose your move: Rock, Paper, or Scissors
2. The computer will randomly select its move
3. Win conditions:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Your score is tracked throughout the game
5. Play as many rounds as you want!
